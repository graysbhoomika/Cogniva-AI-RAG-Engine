# ⬡ Cogniva AI - Enterprise RAG Engine

Cogniva AI is an enterprise-grade Multilingual Knowledge Assistant designed for secure, scalable, and high-precision information retrieval. Built with a focus on production-ready observability and role-based access control.

## 🚀 Key Features
* **Multi-Modal Ready UI:** Cinematic, clean, and distraction-free interface with custom animations.
* **Metadata Categorization:** Efficient document tagging (Tech, HR, Finance) to prevent AI hallucination.
* **Enterprise Security (RBAC):** Role-Based Access Control ensuring secure access to sensitive data.
* **System Telemetry:** Real-time observability dashboard tracking latency and token usage.

## 🛠 Tech Stack
* **LLM Core:** Gemini 2.5 Flash
* **Framework:** LangChain & Streamlit
* **Vector Database:** ChromaDB
* **Embeddings:** HuggingFace `paraphrase-multilingual-MiniLM-L12-v2`

## 📦 How to Run Locally
1. Clone this repository: `git clone https://github.com/graysbhoomika/Cogniva-AI-RAG-Engine.git`
2. Install the required dependencies.
3. Run the application: `streamlit run app.py`
4. Login Credentials:
   * **Manager:** `admin` / `admin123`
   * **Employee:** `employee` / `emp123`
